# Flask-Dashboard
A simple and clean Flask-based dashboard web application with user authentication and task management functionality.

Flask Dashboard Web Application

 Overview

This is a simple and modern dashboard web application built using Python (Flask). It allows users to register, log in, and manage tasks in a clean and user-friendly interface.


 Features

* User registration and login system
* Secure password hashing
* Personal dashboard
* Task management (Add, View, Delete, Mark Complete)
* Flash messages for user feedback
* Responsive and clean UI


Technologies Used

* Python (Flask)
* SQLite
* HTML, CSS
* Bootstrap

How to Run the Project

1. Clone the repository:

git clone https://github.com/Sinokuhle1Spelman/flask-dashboard.git
cd flask-dashboard

2. Create a virtual environment
python -m venv venv
venv\Scripts\activate

3. Install dependencies
pip install -r requirements.txt

4. Run the application
python app.py

5. Open in browser
http://127.0.0.1:5000
